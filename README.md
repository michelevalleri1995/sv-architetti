# SV Architetti

Sito web per lo studio di architettura SV Architetti — Milano.

## Struttura

- **index.html** — Home
- **about.html** — About
- **projects.html** — Elenco progetti
- **contact.html** — Contatti
- **style.css** — Stili condivisi
- **main.js** — Script condiviso
- **img/** — Immagini e cartelle progetti

## Tecnologie

HTML, CSS e JavaScript puri (nessun framework, nessun build tool).

## Avvio

Apri `index.html` nel browser oppure usa un server locale (es. Live Server in VS Code).
